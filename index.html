About Me

(1) I hold a degree in Computer Science from Zagazig University, where I developed a strong foundation in programming, algorithms, and software engineering principles. During my academic journey, I strengthened my problem-solving skills through C++ and data structures, explored database systems and SQL, and gained practical experience with Python for scripting and backend logic development.

I also worked on web development using Python, built 2D and 3D graphics applications with Python and OpenGL, and studied operating system concepts through hands-on experience with both Linux and Windows environments.

On the backend, I architect scalable solutions using PHP, OOP PHP, Laravel, and MySQL, with a strong focus on clean code, performance, and maintainability.
